# Calculator 🧮

A simple and responsive calculator built using HTML, CSS, and JavaScript.

## Features

* Perform basic arithmetic operations

  * Addition (+)
  * Subtraction (-)
  * Multiplication (*)
  * Division (/)
* Clear display functionality
* Decimal number support
* Real-time display updates
* Responsive layout for different screen sizes

## Technologies Used

* HTML
* CSS
* JavaScript

## Concepts Practiced

* DOM Manipulation
* Event Handling
* JavaScript Variables
* User Input Processing
* Mathematical Expression Evaluation
* Responsive Design

## How It Works

Users can click the calculator buttons to create mathematical expressions. The expression is displayed on the screen and evaluated when the "=" button is pressed.
